# Favicon Setup

The project is currently using `favicon.svg` as the favicon. If you want to add a traditional `favicon.ico` file:

1. Create a favicon.ico file (16x16 and 32x32 sizes recommended)
2. Place it in this `public` folder
3. Update `index.html` to reference it:
   ```html
   <link rel="icon" type="image/x-icon" href="/favicon.ico" />
   ```

## Current Setup
- Using SVG favicon: `/favicon.svg` (scalable, modern browsers)
- SVG contains a blue gradient with "S" letter for Sanskruti Associates
